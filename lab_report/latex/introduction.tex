\chapter*{Введение}
\addcontentsline{toc}{chapter}{Введение}

В лабораторной работе №13 рассматривается практическое применение JavaScript при создании единого интерактивного веб-проекта. В работе объединены базовые операции с переменными, массивами и строками, формирование динамического интерфейса, получение данных через API и подготовка приложения к публикации.

В качестве итогового результата разработан тематический сайт ``Витрина аниме-девочек''. Такой формат позволил не только выполнить требования лабораторной работы, но и собрать все программные части в едином визуальном стиле, где каждая секция интерфейса связана с общей аниме-концепцией.
