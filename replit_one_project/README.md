# Lab 13 - One Replit Project

Единый проект для лабораторной 13:

- Урок 1: бонусный баланс;
- Урок 2: чат и поиск;
- Урок 3: карточки персонажей + модальные окна + API;
- При недоступности API автоматически включается демо-режим.

## Запуск в Replit

1. Создай новый `Node.js Repl`.
2. Загрузи в него все файлы из этой папки.
3. Установи зависимости:
   - `npm install`
4. Нажми `Run` (или выполни `npm start`).
5. Открой веб-приложение во встроенном браузере Replit.

## Что скринить для отчета

1. Экран урока 1 (блок с консольным выводом).
2. Экран урока 2 (блок с перепиской и поиском).
3. Экран урока 3 (карточки персонажей).
4. Открытое модальное окно персонажа.
